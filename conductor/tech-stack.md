# Tech Stack

## Core Framework
- **Flutter:** Latest stable version.

## State Management
- **Provider:** For simple and flexible state management.
- **Dynamic Context:** Support for switching between Direct and Group chat modes with associated metadata (isGroupChat, connectionPath).

## Persistence
- **In-Memory:** No local database; messages are stored in memory for this template.

## Libraries
- **Material 3:** Core UI components.
- **font_awesome_flutter:** For extended icon support.
- **flutter_spinkit:** For loading indicators.
- **intl:** For internationalization and date formatting.

## Fonts
- **Roboto:** Primary application font.
- **Noto Color Emoji:** Fallback font for consistent color emoji rendering.

## Testing
- **flutter_test:** For unit and widget testing.
